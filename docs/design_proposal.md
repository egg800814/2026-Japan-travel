# 【設計規格書】高轉換率旅遊網頁介面 (FOMO 導向)

## 1. 設計核心理念

**「這不是一趟旅行，而是你今年最不想錯過的人生切片。」**
利用「沉浸式慾望」與「稀缺性焦慮」雙引擎，讓使用者滑動 3 秒內產生「我現在就想去」的衝動，並在 10 秒內因「怕搶不到」而按下預訂。

## 2. 視覺風格定義 (Visual Identity)

- **配色方案 (Color Palette)：**
  - **Primary (情感誘發):** `#E85D4E` (晚霞橘紅) - 用於限時折扣、主 CTA，帶有急迫感但具溫度。
  - **Secondary (沉浸質感):** `#1E293B` (深邃夜空藍) - 用於背景與高階行程區塊，提升 Premium 質感，區隔一般廉價團。
  - **Accent (信任與靜謐):** `#0EA5E9` (太平洋海岸藍) - 用於資訊圖表、標籤、天氣預報組件。
  - **Background (呼吸感):** `#F8FAFC` (冷霜白) - 確保高對比度與大面積留白。
- **字體建議 (Typography)：**
  - **標題 (Heading):** `Noto Serif TC` (思源宋體) - 傳遞人文感與在地故事的溫度。
  - **內文 (Body):** `Inter` 或 `Noto Sans TC` - 確保 Mobile 端小字號的極致易讀性。
- **攝影指導 (Photography)：**
  - 第一人稱視角 (POV)，拒絕棚拍假笑。範例：從帳篷內向外拍著清境農場的晨霧，或手握咖啡對著花東海岸線的失焦背景。高對比、略帶底片感 (Film Look) 增強真實故事性。

## 3. 關鍵 UI 區塊拆解

- **[標頭與英雄區 (Hero Section)]**
  - **大比例視差滾動 (Parallax Scrolling):** 首屏採滿版高解析實景影片，文字隨滾動緩速浮現，創造「我已經在那裡」的替代感。
  - **Copywriting:** 「錯過這次，下個花季要等 365 天。」
  - **CTA Button:** 懸浮於底部拇指熱區，採用 `#E85D4E`，文案為「即刻入席 (僅剩 2 位)」。

- **[情緒誘發區：為何你不能錯過]**
  - **微型社會認同 (Social Proof):** 區塊右上角微微閃爍紅點「🔥 過去 1 小時有 42 人正在預覽此行程」。
  - **極限損失規避 (Loss Aversion):** 價格旁壓上動態倒數計時「距離早鳥專屬體驗結束僅剩 02:15:39」。

- **[行程亮點模組：在地化細節與痛點解決]**
  - **天氣預測小卡:** 串接 API 顯示「本週末花東天氣：晴朗 22°C，完美符合星空露營條件」。
  - **交通無縫切換 Tab:** 一鍵查看「高鐵轉乘接駁時間」vs「自駕避塞車私房路線」。
  - **暗黑秘境美食地圖:** 互動式深色地圖，僅標註 3 個「在地人排隊美食」，需點擊（互動）才解鎖詳細資訊，創造探索感。

- **[信任與急迫性組件]**
  - **限動化真實評價 (Stories UI):** 模擬 IG 限時動態的圓像 UI，點擊觀看真實短影音：「原本擔心塞車，但跟著管家的避車路線，我們獨棟木屋看見了清境最美日出。— _來自台南的陳小姐_」
  - **誘餌效應對比表 (Decoy Effect):** 將主推方案加上微陰影與漸層外框高亮（如：加價 800 元升級海景房與延遲退房），讓使用者覺得直上頂規 CP 值最高。

## 4. 互動動畫建議 (Micro-interactions)

- **吸入感進場 (Scroll Reveal):** 圖片在滾動進入視窗時，以 `1.1x` 縮放緩慢還原至 `1x` (Ease-out 曲線 0.6s)，增加視覺呼吸。
- **CTA 脈衝動畫 (Pulse Effect):** 倒數計時最後 10 分鐘，CTA 底部產生輕微擴散波紋，無意識催促點擊。
- **無阻礙單手操作 (Mobile-first):** 預訂按鈕與日期篩選條，在頁面向下滑動時，自動吸附 (Sticky) 於螢幕底部安全區 (Safe-area) 邊緣，絕對在拇指自然操作範圍內。
